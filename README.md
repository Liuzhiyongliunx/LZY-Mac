# IPD项目管理小程序

一款专为研发团队设计的打卡与工作规划工具。

## 功能特性

### 1. 每日打卡
- 工作日每天9点打卡提醒（通过微信订阅消息）
- 录入当天工作计划
- 设置任务预计时间
- 评估工作量

### 2. 打卡记录
- 按月查看打卡历史
- 打卡统计数据（打卡天数、计划工时、打卡率）
- 查看每日打卡详情

### 3. 工作规划
- 添加多个工作任务
- 设置任务优先级
- 预估任务工时
- 查看工作量评估

## 使用说明

### 1. 安装依赖

本项目使用微信云开发，无需额外安装依赖。

### 2. 配置云开发环境

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 开通云开发能力
3. 获取云环境ID
4. 在 `app.js` 中替换 `YOUR_CLOUD_ENV_ID`

### 3. 配置订阅消息

1. 在微信公众平台创建订阅消息模板
2. 获取模板消息ID
3. 在 `app.js` 中替换 `YOUR_TEMPLATE_ID`

### 4. 配置AppID

在 `project.config.json` 中替换 `YOUR_APPID` 为你的小程序AppID。

### 5. 导入项目

1. 打开微信开发者工具
2. 导入项目
3. 选择本项目目录
4. 填入AppID
